# 管理后台 - 转移记录 PRD

## 1. 模块说明

转移记录弹窗用于展示管理后台中 BD、代理、主播之间的归属变更记录。支持三种转移类型的记录查看：转移BD、更换BD、更换代理。

业务背景：
- 一个团队只有一个 BD，BD 创建时绑定团队 ID
- 代理和主播创建时选择 BD，自动继承该 BD 的团队 ID
- 转移BD：将原 BD 下的全部代理和主播批量转移至目标 BD（适用于 BD 离职、团队重组）
- 更换BD：将单个代理及其下主播转移至另一个 BD（适用于代理归属调整）
- 更换代理：将单个主播转移至另一个代理（适用于主播归属调整）

路径：管理后台 → 转移记录（弹窗）

---

## 2. 弹窗结构

- 标题：转移记录
- 右上角关闭按钮（点击关闭弹窗）
- 三个 Tab 切换：转移BD、更换BD、更换代理
- 每个 Tab 显示对应类型的转移记录数量
- 内容区域采用卡片式双栏布局，滚动查看
- 弹窗宽度 960px，最大高度 80vh
- 列表仅显示待生效的记录，已取消或已生效的记录不显示

---

## 3. Tab 切换

### 3.1 转移BD

字段（变更前/变更后对称）：

BD UID、BD昵称、所属团队ID

### 3.2 更换BD

字段（变更前/变更后对称）：

代理UID、代理昵称、所属BDUID、所属BD昵称、所属团队ID

### 3.3 更换代理

字段（变更前/变更后对称）：

主播UID、主播昵称、所属代理UID、代理昵称、所属团队ID

以上三种 Tab：
- 变更前卡片左侧蓝色边框，变更后卡片左侧绿色边框
- 变更前后值不同的字段自动标红（#ff4d4f，加粗）
- 每条记录底部显示：生效时间、操作人、取消按钮

---

## 4. 排序

按生效时间倒序排列（最新在前）

---

## 5. 操作

### 5.1 取消

- 每条记录右下角有"取消"按钮
- 点击弹出内联确认弹窗："确认取消该转移记录？"
- 确认后该记录从列表移除
- 点击弹窗外区域关闭弹窗

---

## 6. UI 规范

- 卡片左侧边框：变更前蓝色（#1677ff），变更后绿色（#52c41a）
- 变更值高亮：红色（#ff4d4f），字重 500
- 中间箭头：蓝色圆形背景，白色箭头
- 操作人默认显示：admin
- 生效时间格式：YYYY-MM-DD HH:mm
